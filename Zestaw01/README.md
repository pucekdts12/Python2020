#### Wywołanie
```
python main.py NUMER_ZADANIA [argumenty dla zadania ...]
```
Przykłady:
```
> python main.py 4 2 4
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+


> python main.py 5 -set1 1 2 3 4 5 6 -set2 4 5 6 7 8 9
SET1: {1, 2, 3, 4, 5, 6}
SET2: {4, 5, 6, 7, 8, 9}
ILOCZYN: {4, 5, 6}
SUMA: {1, 2, 3, 4, 5, 6, 7, 8, 9}
```
